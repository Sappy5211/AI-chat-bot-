# New UI design for chat bot

This folder contains the standalone UI design snapshot for the chatbot interface.

## Files

- `index.html` contains the complete HTML, CSS, and JavaScript needed to recreate the UI.

## Run locally

Open `index.html` directly in a browser, or serve it with:

```powershell
python -m http.server 8010
```

Then visit `http://127.0.0.1:8010/`.
